# Data Classification Using AI — DecodeLabs Project 2

A KNN-based classification model built using Python and scikit-learn
as part of the DecodeLabs AI Industrial Training (Batch 2026).

## Pipeline
- Load Iris dataset (150 samples, 4 features, 3 classes)
- Feature scaling using StandardScaler (Mean=0, Variance=1)
- 80/20 Train-Test Split with shuffle
- KNN Classifier (k=5)
- Evaluation via Confusion Matrix and F1 Score
